# 旧裏ポケカ図鑑 — 拡張パック第1弾

ポケモンカードゲーム「旧裏面」第1弾（拡張パック / 1996年10月20日発売 / 全96枚）のまとめ図鑑サイトです。
カードを番号順に一覧し、タップ／クリックで発売時期・イラストレーター・HP・属性・レアリティなどの詳細を表示します。

## 特徴
- スマホは横幅にカード2枚ずつ、PCは最大5列のレスポンシブグリッド
- No.順表示（No.1〜96）
- 属性／レアリティフィルタ・名前検索
- カード詳細モーダル（前後ナビ・URLハッシュ直リンク対応）

## 構成
- `index.html` — ページ本体
- `assets/style.css` — スタイル
- `assets/app.js` — 一覧描画・フィルタ・モーダル
- `data/cards.js` — カードデータ（96枚）

ビルド不要。`index.html` をブラウザで開くだけで動作します。
